# Copilot Instructions for Setup Project

## Project Overview
This is a local-only Ansible playbook for setting up a user's environment on a Debian-based system.

## Key Files
- `setup.yml` - Main Ansible playbook
- `dotfiles/` - Dotfile configurations managed by the playbook
- `ansible.cfg` - Ansible configuration
- `.ansible-lint` - Linting configuration for Ansible

## Development Workflow

### Prerequisites
1. Install [`asdf`](https://asdf-vm.com/guide/getting-started.html)
2. Add and install plugins:
   ```bash
   asdf plugin add uv
   asdf install
   ```

### Common Tasks
- **Lint code**: `uvx ansible-lint --strict --fix=all`
- **Check playbook**: `uvx --from 'ansible>=12' ansible-playbook --ask-become-pass setup.yml --check`
- **Run specific tags**: `uvx --from 'ansible>=12' ansible-playbook --ask-become-pass setup.yml --tags <tag> --check`
- **Check tool versions**: 
  - `uvx --from 'ansible' ansible --version`
  - `uvx ansible-lint --nocolor --version`

## Code Style
- Follow Ansible best practices
- Use consistent YAML formatting
- Include descriptive names and comments for tasks

## CI/CD
Linting is automatically run by the CI pipeline on pull requests.
